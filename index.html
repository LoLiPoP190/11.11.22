Git Instructions

How To Create local repository

  1. Right click IT folder using the mouse
  2. Select "Git Bash Here"
  3. Execute "mkdir FolderName"
  4. Execute "cd FolderName"
  5. Execute "git init"

How To Link repo with GitHub

  1. Go to new repo page
  2. Fill repo name fieald and press "Create a new repository" button
  3. Execute "git remote add origin https://github.com/LoLiPoP190/FolderName.git"
  4. Execute "git branch -M main
  5. Execute "git push -u origin main"
